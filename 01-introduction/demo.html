Méthode 1 : CSS Externe (ce fichier)

Les styles de cette page sont définis dans le fichier demo.css, lié dans le <head> avec la balise <link>. C'est la méthode recommandée dans la grande majorité des cas.

<link rel="stylesheet" href="demo.css">
