---

## 📖 À propos de ce cours

Ce dépôt contient une **formation CSS complète** organisée comme un vrai projet pédagogique professionnel. Chaque module est documenté en détail, accompagné d'exemples pratiques, d'exercices et de corrigés.

### 🎯 Objectifs d'apprentissage

À la fin de ce cours, vous serez capable de :

- ✅ Comprendre les fondamentaux du CSS et son rôle dans le développement Web
- ✅ Styliser des pages Web modernes et professionnelles
- ✅ Maîtriser les systèmes de mise en page (Flexbox, Grid)
- ✅ Créer des interfaces utilisateur professionnelles
- ✅ Construire des designs responsive (adaptatifs)
- ✅ Optimiser les performances CSS
- ✅ Utiliser les techniques CSS avancées

## 🚀 Comment utiliser ce cours

### Étape 1 : Cloner le dépôt

```bash
git clone https://github.com/Mamby-DOUMBIA/cours-complet-css.git
cd cours-complet-css
```

### Étape 2 : Ouvrir dans VS Code

```bash
code .
```

### Étape 3 : Installer l'extension Live Server

Dans VS Code, installez l'extension **Live Server** (ritwickdey.liveserver) pour prévisualiser vos fichiers HTML en temps réel.

### Étape 4 : Suivre les modules dans l'ordre

Commencez par le module `01-introduction` et progressez dans l'ordre.
